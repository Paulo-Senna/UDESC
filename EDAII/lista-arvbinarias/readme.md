# Exercicios

### 1. Implemente um algoritmo que visite todos os noes de uma arvore binaria com uma busca em profundidade in-order, pre-order e pos-order

Exemplo, para a arvore abaixo a saida deve ser:

* _Pre-order:_ 4, 2, 1, 3, 8, 6, 5, 7, 9
* _In-order:_ 1, 2, 3, 4, 5, 6, 7, 8, 9
* _Pos-order:_ 1, 3, 2, 5, 7, 6, 9, 8, 4
